# Raj True Dent - Clinic OPD Console

A comprehensive, modern Dental Clinic Management System designed to streamline daily operations, patient management, and treatment tracking. Built with a focus on aesthetics, performance, and user experience.

## 🚀 Features

-   **🔐 Premium Secure Access**:
    -   Glassmorphism-styled login panel with secure authentication flow.
    -   Session persistence for seamless user experience.

-   **📊 Dynamic Dashboard**:
    -   Real-time overview of clinic statistics.
    -   Quick access to daily appointments and pending tasks.

-   **👥 Patient Management**:
    -   Complete patient records with search and filter capabilities.
    -   Detailed history tracking for every patient.

-   **📅 Appointment Scheduling**:
    -   Efficient appointment booking system.
    -   Calendar view for better time management.

-   **🦷 Clinical Modules**:
    -   **RCT Module**: Specialized tracking for Root Canal Treatments.
    -   **Pulpectomy**: Dedicated pediatric dental care tracking.
    -   **Crown / Lab**: Management of dental crowns and lab work status.
    -   **Orthodontics**: Progress tracking for ortho patients.

-   **💰 Billing & Payments**:
    -   Integrated billing system with payment tracking.
    -   Auto-allocation of payments to pending treatments.

## 🛠️ Tech Stack

-   **Frontend**: React 18, TypeScript
-   **Build Tool**: Vite
-   **Styling**: Tailwind CSS (Custom Configuration)
-   **Icons**: Lucide React
-   **State Management**: React Context API
-   **Notifications**: React Hot Toast

## 🏁 Getting Started

### Prerequisites

-   Node.js (v16 or higher)
-   npm or yarn

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/iamdurgashankar/Clinic-OPD-console.git
    cd Clinic-OPD-console
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run the development server**
    ```bash
    npm run dev
    ```

4.  **Build for production**
    ```bash
    npm run build
    ```
